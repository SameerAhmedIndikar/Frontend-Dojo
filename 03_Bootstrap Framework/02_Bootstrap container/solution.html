Bootstrap Containers

Containers are the fundamental building blocks of Bootstrap's layout system.

They provide a means to center and horizontally pad your site's contents.

Types of Containers

  • .container: Responsive fixed width container.
  • .container-fluid: Full width container, spanning the entire width of the viewport.
  • .container-{breakpoint}: Responsive containers for each breakpoint (e.g., .container-md).
