DOM-Document Object Model

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as a tree of nodes, where each node is an object representing a part of the document.

Key Concepts

  • Document: The entire HTML or XML document.
  • Element: An individual HTML or XML tag (e.g., <div>, <p>).
  • Node: Any single point in the document tree, including elements, text, and attributes.
  • Tree Structure: The hierarchical representation of the document, where elements can contain other elements.

Why is the DOM Important?

The DOM allows developers to create dynamic and interactive web pages. By manipulating the DOM using JavaScript, developers can change the content, structure, and style of a web page in response to user actions or other events.

Pillars of DOM

There are mainly 4 pillars of DOM

  • Structure
  • Traversal & Navigation
  • Manipulation
  • Events
