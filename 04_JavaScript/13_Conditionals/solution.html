If Statement

The if statement is used to execute a block of code only if a specified condition is true. It evaluates the condition inside the parentheses and runs the code inside the curly braces if the condition is met.

Example: If the age is greater than 18, display "You are an adult".

If-Else Statement

The if-else statement provides an alternative path. If the condition in the if statement is true, the code in the if block executes; otherwise, the code in the else block executes.

Example: If the temperature is above 30Â°C, display "It's hot"; else, display "It's not hot".

Else-If Ladder

The else-if ladder allows checking multiple conditions sequentially. If the first condition is false, it checks the next else-if condition, and so on. If none are true, the else block (if present) executes.

Example: Check grades - if score >= 90, "A"; else if score >= 80, "B"; else if score >= 70, "C"; else "F".

Switch Statement

The switch statement evaluates an expression and matches it against multiple case values. When a match is found, the corresponding code block executes. It's useful for multiple conditions based on a single value.

Example: Based on day number (1-7), display the day name using cases for each day.
