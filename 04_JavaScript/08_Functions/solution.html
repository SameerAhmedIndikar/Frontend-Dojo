Functions in JavaScript

A function is a block of code designed to perform a particular task. It can take inputs (parameters) and return outputs.

Parameters and Arguments

Parameters are placeholders in the function definition.

Arguments are the actual values passed to the function when calling it.

Types of Functions

1. Named Function

Declared with a name using the 'function' keyword.

2. Anonymous Function

A function without a name, often assigned to a variable.

3. Arrow Function

A shorter syntax for functions, using '=>'.
