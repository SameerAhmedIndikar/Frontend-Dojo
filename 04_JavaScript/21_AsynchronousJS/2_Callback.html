Callbacks in Asynchronous JavaScript

What is a Callback?

A callback is a function passed into another function as an argument, which is then invoked inside the outer function to complete some kind of routine or action. Callbacks are a fundamental way to handle asynchronous operations in JavaScript.

They allow you to run code after an asynchronous task has completed, such as after fetching data or waiting for a timer.

Basic Callback Function Example

                
                    function greet(name, callback) {
                        console.log('Hello ' + name);
                        callback();
                    }
                    function sayGoodbye() {
                        console.log('Goodbye!');
                    }
                    greet('Sameer', sayGoodbye);

In this example, the greet function takes a name and a callback function as arguments. After greeting the person, it calls the sayGoodbye function.

Callback Hell Example

                
                    function task(name, callback){
                        setTimeout(()=>{
                            console.log(name);
                            callback();
                        }, 1000);
                    }
                    task('Step 1', ()=>{
                        task('Step 2', ()=>{
                            task('Step 3', ()=>{
                                task('Step 4', ()=>{
                                    .............
                                    task('Steps 100', ()=>{
                                        console.log('100 Steps completed');
                                    });
                                });
                            });
                        });
                    });

This nested structure is often called "callback hell" or "pyramid of doom" because it becomes difficult to read and maintain.
